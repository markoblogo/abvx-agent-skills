# Skill Evolution Source Registry

Curated sources for future bounded skill evolution. A source is eligible only when it can produce a reusable mechanism and a validation path.

## Active Sources

| Source | Use | Status |
|---|---|---|
| https://arxiv.org/abs/2605.23904 | SkillOpt-style bounded skill edits and accepted/rejected edit discipline | accepted-reference |
| https://arxiv.org/abs/2510.01171 | Diversity-first candidate generation before validation | accepted-reference |
| https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering | Context degradation, filesystem context, tool contracts, bounded evaluation, researcher-loop structure | accepted-reference |
| https://github.com/cobusgreyling/loop-engineering | Loop readiness, cadence, state, isolation, verifier, budget, run logs, and human gates | accepted-reference |

## Exclusion Rules

- Do not vendor whole skill packs by default.
- Do not promote a source when it only restates rules already present.
- Do not accept self-improvement mechanisms without human review and validation artifacts.
