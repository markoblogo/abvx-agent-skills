# Skill Card: frontend-product-builder

## Description
Guides frontend implementation toward usable, brand-aware, responsive, verified product experiences.

## Owner
ABVX / Anton Biletskiy-Volokh

## License
MIT. See repository LICENSE.

## Intended Use
Use for apps, landing pages, pitch pages, dashboards, prototypes, redesigns, and frontend polish passes.

## Out of Scope
Do not use to create decorative mockups that lack usable workflow, verification, or responsive behavior.

## Sources and Attribution
ABVX adapted from local frontend skill practice.

## Inputs and Outputs
Inputs: repository, brief, brand kit, screenshots, design references, existing components.

Outputs: frontend changes, responsive UI, asset usage, browser verification notes.

## Risks and Mitigations
- Risk: generic design. Mitigation: define visual thesis and first-screen job.
- Risk: mobile breakage. Mitigation: verify common viewports and overflow.
- Risk: inaccessible controls. Mitigation: include focus, labels, states, and semantic controls.

## Evaluation
Evaluated by structural validation and manual review against frontend implementation workflows.
